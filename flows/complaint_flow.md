# Complaint Flow

1. **User files complaint**
   - Capture booking ref (if applicable), incident time, location, and short description.

2. **Immediate response**
   - Acknowledge and apologize.
   - Provide immediate safety instructions if needed.

3. **Log & escalate**
   - Save structured complaint (type, severity, evidence).

4. **Follow-up**
   - Tell user expected timeline for update.
